# Asset Attribution

This directory will contain game assets sourced from:

- Diablo II original game assets (Blizzard Entertainment)
- Fan-made recreations
- Public domain alternatives

All assets must comply with fair use or be properly licensed.

### AI-generated assets
The `generated/` subfolder contains images generated by the `image-gen`
skill via [Pollinations.AI](https://pollinations.ai). Provenance for every
file (prompt, seed, model, sha256, timestamp) is recorded in
`generated/manifest.json`. See `docs/art/style-guide.md` for the style
contract and `docs/art/seed-registry.md` for canonical subject seeds.
The art-director agent is the single owner of style consistency and must
approve every generation request before it is run.
